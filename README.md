## ▶️ How to Use

Clone the repository:

```bash
git clone <your-repository-url>
```

Navigate into the folder:

```bash
cd Bash_Projects
```

Open a project folder:

```bash
cd project-1
```

Run a Bash script:

```bash
bash script-name.sh
```
